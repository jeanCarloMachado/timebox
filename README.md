# timebox

Control your time in boxes. The live website is [here](https://jeancarlomachado.net/timebox/)

## Setup

### Dependencies

 - leiningen
 - clojurescript

To get an interactive development environment run:

    make

and open your browser at [localhost:3449](http://localhost:3449/).
